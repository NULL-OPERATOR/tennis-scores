# TENNIS TEST TES :palm_tree:
##### scoring for a tennis game with redux.

scores a *game* (not set or match), also runs on node 8.9.0

[tennis rules][1]


clone & install
```
$ git clone [repo]
$ npm i
```

to run the dev server
```
$ npm start
```
build and run the node server
```
$ npm build
$ node server
```



- node: v8.9.0
- express: v4.16.2  
- react: v16.0.0  
- redux: v3.7.2
